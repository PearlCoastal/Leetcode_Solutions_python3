## 思路

如何用我会的数据结构解决一道 bitwise。

## 代码
```python
class Solution:
    def singleNumber(self, nums: List[int]) -> int:

        dic = collections.defaultdict(int)

        for num in nums:
            dic[num] += 1
        
        for key, value in dic.items():
            if dic[key] == 1:
                return key
```

## 复杂度分析
- 时间复杂度： O(n)
- 空间复杂度： O(n)
