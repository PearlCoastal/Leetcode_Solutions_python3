LC.516 最长回文子序列

## 思路

子序列 --> 动态规划嘛。

回文字符串主要看两头， 如果中间已经是一个回文了， 那两头再相等， 长度就等于 中间的长度 + 2。

两头要是不相等， 那就取两端子序列的最大值。

（ @cache 算作弊嘛...）

## 代码
```python
class Solution(object):
    def longestPalindromeSubseq(self, s):
        @cache
        def dp(left, right):
            if left == right:
                return 1
            elif left > right:
                return 0
            if s[left] == s[right]:
                return dp(left + 1, right - 1) + 2
            return max(dp(left + 1, right), dp(left, right - 1))

        return dp(0, len(s) - 1) 

```
